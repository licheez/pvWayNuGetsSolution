How to create a dotNet framwork nuGet package
=============================================

1. Create project of type dotNet framework class library (for example 4.6.1)
2. Edit AssemblyInfo.cs (under Properties)

	[assembly: AssemblyTitle("MethodResultWrapper")]
	[assembly: AssemblyDescription("Provides a generic wrapper that returns whether or not a method succeeded of failed carrying the method result on success or a list of notifications/errors in case of failure ")]
	[assembly: AssemblyConfiguration("")]
	[assembly: AssemblyCompany("pvWay")]
	[assembly: AssemblyProduct("MethodResultWrapper")]
	[assembly: AssemblyCopyright("Copyright pvWay Ltd Â©  2020")]
	[assembly: AssemblyTrademark("")]
	[assembly: AssemblyCulture("")]

3. Unload the project (needed in Vs2017 and <)
4. Right-click the project as select Edit csproj
5. Change the default configuration from Debug to Release

	from    <Configuration Condition=" '$(Configuration)' == '' ">Debug</Configuration>
	to		<Configuration Condition=" '$(Configuration)' == '' ">Release</Configuration>

6. Reload the project (needed in Vs2017 and <)
7. Open the project Properties
8. in the "Application" tab set the Default namespace to something unique
	Default namespace: pvWay.MethodResultWrapper

9. Download nuget.exe (version v5.2.0) from https://www.nuget.org/downloads (Warning: there is a bug in version above 5.2.0) that creates a issue when packing 
	=> name the downloaded file "nuget5.2.0.exe"
10. Copy the "nuget5.2.0.exe" file within the project folder <github_repo>\MethodResultWrapper
11. Open a command promt from the project folder <github_repo>\MethodResultWrapper
12. Create the nuget specification file for the project using the command
	"nuGet5.2.0.exe spec MethodResultWrapper.csproj"
	=> this create the "MethodResultWrapper.nuspec" file
13. Back in Visual studio edit the nuspec file to:

	<?xml version="1.0"?>
	<package >
	  <metadata>
		<id>$id$</id>
		<version>$version$</version>
		<title>$title$</title>
		<authors>Pierre Van Wallendael</authors>
		<owners>pvWay</owners>
		<projectUrl>http://www.pvWay.com</projectUrl>
		<iconUrl>http://www.pvway.com/images/Logo150x200px.png</iconUrl>
		<requireLicenseAcceptance>false</requireLicenseAcceptance>
		<description>Provides a generic wrapper that returns whether or not a method succeeded of failed carrying the method result on success or a list of notifications/errors in case of failure</description>
		<releaseNotes>Summary of changes made in this release of the package.</releaseNotes>
		<copyright>Copyright pvWay 2020</copyright>
		<tags>pvWay MethodResult Logger</tags>
	  </metadata>
	</package>

14. For packing the project run the command "nuGet5.2.0.exe pack" from the project folder
